2. Description of Service

aLLMost is a Chrome browser extension that provides pre-send intelligence for large language model (LLM) chat interfaces. It shows you similar past prompts before you send a new one (Déjà Prompt), recommends which LLM is best suited to your current task (Model Router), and operates on supported LLM sites including Claude, ChatGPT, Gemini, DeepSeek, Kimi, and Grok. A paid "Pro" tier offers additional features including evasion detection, contradiction tracking, a cross-session dashboard, and unlimited captured-prompt history.
